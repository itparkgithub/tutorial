Hello here are the steps to using github or gitlab

  • create account
  • open gmail for verification code
  • create a new (empty) repo and name it correctly
  • Click HERE to download git bash
  • git bash go to desired directory and clone the repo created in step 2
  • cd said repo and explain branches
  • switch branch using code: git checkout -b (desired branch name here)
  • check the status of the branch using code: git status
  • add the desired files or all files to the git using code: git add (file name) | (use . for adding all files or edits)
  • check the status of the branch again using code: git status to see if the files have been added correctly (it will highlight green)
  • commit code with a message for reference later down the road using code: git commit -m "(input message here)"
  • push your code using code: git push origin (current branch name)
